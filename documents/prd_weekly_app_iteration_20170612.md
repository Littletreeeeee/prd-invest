### 1. 首页焦点图

首页焦点图轮播逻辑：

* 自动滚动：每5秒向下一张滚动
* 自动滚动：最后一张焦点图的下一张，为第一张焦点图，循环滚动
* 若用户手动滚动，则自动滚动暂停10秒

首页焦点图元素：

|焦点图元素|类型|可以为空|说明|
|:---:| :-----------: | :-----------: | :-----------: |
|主标题|文本|是||
|副标题|文本|是||
|焦点图封面|图片|否||
|点击跳转|对象|否|具体跳转情况见下|

&emsp;&emsp;  

焦点图不同荐类型跳转页面：

|推荐类型|跳转页面|
|:---:| :----------- |
|普通文章|文章详情页|
|杂志|杂志目录页|
|专题|专题详情页|
|单行本|已购买用户-单行本详情页；未登录、未购买用户-单行本说明页|

&emsp;&emsp;

#### 2.10 单行本说明页、单行本详情页进入逻辑

已下均为案例：

##### 2.10.1 未登录用户

1）用户点击首页焦点图、单行本模块焦点图、全部单行本 -> 进入单行本说明页 -> 点击试读 -> 进入试读列表 -> 查看文章 -> 点击底部单行本订阅引导 -> 跳转登录页 ->

登录后若未购买：弹出购买框 -> 购买后跳转单行本详情页 -> 点击返回，回到全部单行本页

登录后若已购买：跳转单行本详情页 -> 点击返回，回到全部单行本页

2）用户点击首页焦点图、单行本模块焦点图、全部单行本 -> 进入单行本说明页 -> 点击订阅 -> 跳转登录页 ->

登录后若未购买：弹出购买框 -> 购买后跳转单行本详情页 -> 点击返回，回到全部单行本页

登录后若已购买：跳转单行本详情页 -> 点击返回，回到全部单行本页

3）用户点击首页焦点图、单行本模块焦点图、全部单行本 -> 进入单行本说明页 -> 点击会员折扣提示 -> 跳转登录页 ->

登录后若未购买：弹出会员购买框 -> 购买后刷新单行本详情页价格 -> 点击返回，回到全部单行本页

登录后若已购买：跳转单行本详情页 -> 点击返回，回到全部单行本页

4）在作者详情页、搜索结果页、相关文章推荐中点击单行本文章（试读&非试读）-> 点击订阅 -> 跳转登录页 ->

登录后若未购买：弹出购买框 -> 购买后跳转单行本详情页 -> 点击返回，回到全部单行本页

登录后若已购买：跳转单行本详情页 -> 点击返回，回到全部单行本页

##### 2.10.2 登录未购买用户

1）用户点击首页焦点图、单行本模块焦点图、全部单行本 -> 进入单行本说明页 -> 点击试读 -> 进入试读列表 -> 查看文章 -> 点击底部单行本订阅引导 -> 弹出购买框 -> 购买后跳转单行本详情页 -> 点击返回，回到全部单行本页

2）用户点击首页焦点图、单行本模块焦点图、全部单行本 -> 进入单行本说明页 -> 点击订阅 -> 弹出购买框 -> 购买后跳转单行本详情页 -> 点击返回，回到全部单行本页


3）用户点击首页焦点图、单行本模块焦点图、全部单行本 -> 进入单行本说明页 -> 点击会员折扣提示 -> 弹出会员购买框 -> 购买后刷新单行本详情页价格 -> 点击返回，回到全部单行本页

4）在作者详情页、搜索结果页、相关文章推荐中点击单行本文章（试读&非试读）-> 点击订阅 -> 弹出购买框 -> 购买后跳转单行本详情页 -> 点击返回，回到全部单行本页

#### 2.10.3 登录已购买用户

1）用户点击首页焦点图、单行本模块焦点图、全部单行本 -> 进入单行本详情页 -> 点击右上角"..."，选择查看单行本说明 -> 进入单行本说明页（无订阅、试读按钮）

2）在作者详情页、搜索结果页、相关文章推荐中点击单行本文章（试读&非试读）-> 进入文章完整详情页

#### 2.11 单行本文案调整
